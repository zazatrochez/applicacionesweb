Aplicación Móvil FitNav

Aplicación Android desarrollada en Android Studio para fitness y entrenamiento. Incluye sistema completo de navegación, gestión de WODs y ejercicios Tabata con interfaz intuitiva.

Android Studio Java Activities RecyclerView Intents
Características principales:
  • MainActivity: Interfaz principal con botones de Iniciar Sesión y Registrar
  • Panel de Control: Navegación centralizada a todas las funcionalidades
  • WodsActivity: Lista dinámica de WODs usando RecyclerView
  • TabatasActivity: Gestión de ejercicios Tabata con RecyclerView
  • Sistema de Navegación: Intents para transición fluida entre activities
